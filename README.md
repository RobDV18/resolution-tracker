# Resolution Tracker (Web PWA)

Simple, client-only Progressive Web App to log daily resolution activities (paint, workout, drawing, etc.). Data is stored in `localStorage`. Works offline once installed.

How to run locally:
1. Put the files into a folder.
2. Serve with a static server (example using Python):

```bash
# Python 3
python -m http.server 8000
# then open http://localhost:8000 on your phone or desktop
